Endpoints disponibles
POST /api/analyze
Analyser un bail PDF ou texte. Retourne le score, les clauses et les corrections.
GET /api/analyses
Lister toutes les analyses de l'agence avec filtres (statut, date, type).
GET /api/analyses/:id
Récupérer le rapport complet d'une analyse (score, clauses, corrections, PDF).
POST /api/bail/generate
Générer un bail conforme selon les paramètres fournis. Retourne le texte et le PDF.
Authentification
Authorization: Bearer bs_live_votre_cle_api
Content-Type: application/json
